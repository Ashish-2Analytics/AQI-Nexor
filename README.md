## 📌 Project Overview
This project provides a **data-driven deep dive** into air quality across six major global cities. By processing a high-frequency dataset of **52,560+ records**, I identified critical pollution patterns, seasonal trends, and the primary drivers behind hazardous **AQI (Air Quality Index)** levels. 📉💨

<br>

## 🚀 Key Highlights
<table>
  <tr>
    <td><b>🔢 Massive Scale</b></td>

  </tr>
  <tr>
    <td><b>📍 Urban Mapping</b></td>
